# Task 1 - Ray Tracing

## Steps to Run

1. Navigate to the project directory:
   cd "Project 1/Task 1"
2. Compile the program:
   javac -d bin src/RayTracing.java
3. Run the program:
   java -cp bin RayTracing
4. an output image will be created under Task 1 folder
